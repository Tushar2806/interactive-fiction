Take a moment to think about what the code might be before you start entering numbers.

Enter the 8 digit code to enter the house

The buttons on the keypad are labelled "enter," "retry," and the numbers 0 through 9. To enter your code, press the buttons. Before clicking enter, you can always hit the "retry" button to start over.

The lock screen will disappear if you enter the right code, allowing you to access the secured area.

If you enter an incorrect code, the lock screen will display a message telling you that the code is incorrect and you will fall into the basement

YOU ONLY HAVE ONE CHANCE TO GET THE CODE RIGHT TO OPEN THE LOCK.
